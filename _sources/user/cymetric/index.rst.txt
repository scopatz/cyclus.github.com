Cymetric
==========

Cymetric (pronounced *symmetric*) is a general analysis library and tool for 
interacting with Cyclus data. Its primary purpose is to compute and display 
fuel cycle metrics while making those metrics easy to write.

The cymetric project repository is located at http://github.com/cyclus/cymetric.

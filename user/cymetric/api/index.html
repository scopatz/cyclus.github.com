Cymetric APIÂ¶

The following modules make up the cymetric package. Note that most of the important stuff you can get directly from cymetric without needing to import the module. For example, the following two imports are the same:

from cymetric import metric           # nice and short
from cymetric.metrics import metrics  # where 'metric' really lives

Interface Modules:

  • cymetric.schemas - Schemas
  • cymetric.metrics - Metrics
  • cymetric.root_metrics - Root Metrics
  • cymetric.evaluator - Evaluators
  • cymetric.execution - Execution

Helper Modules:

  • cymetric.main - Command Line Interface
  • cymetric.tools - Tools
